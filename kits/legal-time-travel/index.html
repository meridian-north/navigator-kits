⚖ For Counsel — three things to know before the demo
I
The sources are the sources. Every event on this timeline traces to a primary source — enacted statutory text, official amendment, HHS/agency declaration, or published court opinion. The corpus indexes what sovereign authority actually said and when it said it. No inference, no editorial gloss.
II
Proclaimed WHY is a schema field, not a footnote. When Congress or an agency embedded its rationale in the instrument itself — findings clauses, purpose statements, legislative declarations — that text is indexed as a first-class data point. You can query it, filter on it, and cite it directly. The gold highlights on the timeline mark these moments.
III
Every output is corpus-version pinned and reproducible. Each corpus release carries a SHA-256 manifest. The same query against the same version returns the same result — the way a certified copy is the same as the original. If any output from this system were needed in a proceeding, the exact corpus version and inputs can be produced and re-run by any party.
